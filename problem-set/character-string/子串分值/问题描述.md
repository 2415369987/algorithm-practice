### 题目描述

对于一个字符串 $S$，我们定义 $S$ 的分值 $f(S)$ 为 $S$ 中恰好出现一次的字符个数。例如 $f(aba)=1,f(abc)=3,f(aaa)=0$。

现在给定一个字符串 $S_{0⋯n−1}$（长度为 $n$，
$1≤n≤10^5$），请你计算对于所有 $S$ 的非空子串 $S_{i⋯j(0≤i≤j<n)}$，
$f(S_{i⋯j})$ 的和是多少。

### 输入描述

输入一行包含一个由小写字母组成的字符串 $S$。

### 输出描述

输出一个整数表示答案。

### 输入输出样例

#### 示例

> 输入

```txt
ababc
```

> 输出

```txt
21
```

### 运行限制

- 最大运行时间：1s
- 最大运行内存: 256M
